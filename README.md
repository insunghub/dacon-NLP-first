
### 💡 데이터셋 및 증강 전략 (Data & Augmentation)

* 한국어 텍스트 분류의 표준 벤치마크 데이터셋인 YNAT(연합뉴스 뉴스 토픽)을 활용하여 도메인 특화 문체에 대한 모델의 기초 체력을 강화했습니다.
* 규칙 기반 및 가상 데이터 생성 알고리즘(`generate_mock_data.py`)을 설계하여 소수 클래스 데이터를 증강(Augmentation)하고 오버샘플링함으로써 데이터 불균형 문제를 해소하는 파이프라인을 구축했습니다.

### 💡 텍스트 최적화 및 EDA (Text Optimization)

* 철저한 탐색적 데이터 분석(EDA)을 통해 입력 데이터의 텍스트 길이 분포를 분석했습니다.
* 정보 손실을 최소화하면서 연산 효율을 극대화하는 최적의 문장 수용 길이(**`max_length=512`**)를 도출하여 전처리에 적용했습니다.

### 💡 4단계 모듈러 학습 스크립트 구현 (Model Training)

* 전체 파이프라인의 구조적 안정성과 리스크 분산을 위해 체계적인 4단계 모듈러 학습 프로세스를 확립했습니다.
1. **`파이프라인 검증`:** 경량 모델 기반 베이스라인 코드로 엔드투엔드 파이프라인 무결성 검증
2. **`한국어 특화`:** 어휘 사전(Vocab) 및 구어체 이해도가 검증된 `KoELECTRA-v3-base` 모델 가동
3. **`체급 확장`:** 깊은 맥락 추론을 위해 Gradient Accumulation 및 FP16 연산을 적용한 `RoBERTa-Large` 모델 가동
4. **`통합 앙상블`:** 개별 모델의 예측 확률(Softmax)을 결합하여 예측 편향을 상쇄하는 소프트 보팅(Soft Voting) 마스터 스크립트 구현



### 💡 엄격한 검증 시스템 구축 (Validation Setup)

* Public 리더보드에만 모델이 맞춤 설정되어 실제 평가(Private 리더보드) 시 순위가 폭락하는 현상을 방지하고자 했습니다.
* 이를 위해 데이터의 카테고리 비율을 균등하게 보존하며 교차 검증을 수행하는 **Stratified 5-Fold 모듈**을 설계하여 견고한 모델 신뢰성을 확보했습니다.
